SETUP (One Time)
===============================================================================

1. Edit .env file with your Amazon credentials:
   AMAZON_CLIENT_ID=...
   AMAZON_CLIENT_SECRET=...
   AMAZON_REFRESH_TOKEN=...

2. You're done! Environment and dependencies are already installed.

===============================================================================
BASIC USAGE
===============================================================================

# Download all reports (last 7 days)
conda run -n glancefive python glancefive.py

# With specific date range
conda run -n glancefive python glancefive.py --start-date 2026-05-01 --end-date 2026-05-09

# Specific reports only
conda run -n glancefive python glancefive.py --reports spTargeting spCampaigns

# Save results to JSON
conda run -n glancefive python glancefive.py --save-results

===============================================================================
ENVIRONMENT COMMANDS
===============================================================================

# See all environments
conda info --envs

# Activate GlanceFive environment
conda activate glancefive

# Deactivate environment
conda deactivate

# Install more dependencies (if needed)
conda run -n glancefive pip install <package_name>

===============================================================================
TROUBLESHOOTING
===============================================================================

ERROR: "Client authentication failed"
→ Check .env file has valid Amazon credentials

ERROR: "No profile found"
→ Verify profile ID exists in Amazon Seller Central

ERROR: "Network error"
→ Check internet connection and API endpoints

REPORT EMPTY: Sponsored Display (SD) reports
→ Expected - SD often has no data
